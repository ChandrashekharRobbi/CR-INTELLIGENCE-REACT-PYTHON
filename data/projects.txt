AI Resume Portfolio — RAG Powered Assistant System
Overview

AI Resume Portfolio is a multi-page intelligent portfolio application built using Retrieval Augmented Generation (RAG) architecture. It includes a context-aware AI assistant capable of answering questions about experience, skills, projects, and architecture using semantic search and vector retrieval.

The system combines document ingestion, embedding generation, vector indexing, and LLM inference to produce accurate, context-aware responses.

Problem Solved

Traditional portfolios are static and cannot dynamically answer questions about experience or technical depth.

This system transforms a portfolio into an interactive AI assistant that:

Understands natural language queries

Retrieves relevant context from indexed data

Generates structured, intelligent responses

Explains system architecture and technical decisions

Architecture Overview

Document ingestion from structured text files

Embedding generation using transformer-based models

Vector index creation and local storage

Semantic similarity search

Context injection into LLM

Groq-powered inference for fast response generation

Session-based chat memory
